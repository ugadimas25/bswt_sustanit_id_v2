## Design Overview

### Platform Purpose
Bentang Sawit is a sustainability and traceability platform for Indonesian palm oil smallholder farmers. The platform must:
- Support users with varying digital literacy levels
- Work on low-end Android devices with limited connectivity
- Display data-dense dashboards clearly
- Support bilingual content (Indonesian primary, English metadata)
- Function offline with service worker caching

### Design Philosophy
**Material Design 3** adapted for agricultural context with focus on:
- **Clarity Over Complexity**: Every element serves a purpose
- **Icon-First Communication**: Visual symbols supplement text for accessibility
- **Data Legibility**: Charts, maps, and metrics must be instantly comprehensible
- **Progressive Disclosure**: Essential info first, details on demand
- **Environmental Connection**: Design evokes tropical plantations and stewardship

### Typography

#### Font Stack
```css
--font-sans: 'Space Grotesk', sans-serif  /* Primary - modern, legible */
--font-serif: 'Geist', sans-serif         /* Secondary - technical */
--font-mono: 'Geist Mono', monospace      /* Data tables, code */

/* Fallback for Indonesian */
font-family: 'Noto Sans', 'Roboto', sans-serif;
```

#### Type Scale
```css
/* Page Titles */
text-3xl md:text-4xl font-bold           /* 1.875rem / 2.25rem */

/* Section Headers */
text-xl font-semibold                     /* 1.25rem */

/* Card Titles */
text-lg font-medium                       /* 1.125rem */

/* Body Text */
text-base leading-relaxed                 /* 1rem, 1.625 line-height */

/* Captions */
text-sm                                   /* 0.875rem */

/* Data Labels */
text-xs uppercase tracking-wide           /* 0.75rem */
```

#### Text Styling Guidelines
- **Headings**: Bold weight, dark foreground color
- **Body**: Regular weight, slightly increased line-height for readability
- **Labels**: Medium weight, uppercase for data labels
- **Links**: Underline on hover, primary color
- **Muted Text**: Use `text-muted-foreground` for secondary information

### Spacing System

#### Base Unit: 0.25rem (4px)
```css
gap-2   /* 0.5rem  - 8px  - tight spacing */
gap-3   /* 0.75rem - 12px - compact */
gap-4   /* 1rem    - 16px - default */
gap-6   /* 1.5rem  - 24px - comfortable */
gap-8   /* 2rem    - 32px - generous */
gap-12  /* 3rem    - 48px - section breaks */
```

#### Component Padding
- **Mobile**: `p-6` (1.5rem / 24px)
- **Desktop**: `p-8` (2rem / 32px)
- **Section Gaps Mobile**: `space-y-8`
- **Section Gaps Desktop**: `space-y-12`
- **Card Grids Mobile**: `gap-4`
- **Card Grids Desktop**: `gap-6`
- **Form Fields**: `space-y-4`

### Border Radius
```css
--radius: 0rem  /* Sharp corners for modern aesthetic */

/* Component-specific */
rounded-md  /* 0.375rem - 6px  - buttons, inputs */
rounded-lg  /* 0.5625rem - 9px - cards */
rounded-xl  /* 0.75rem - 12px  - large containers */
```

### Elevation & Shadows
```css
/* Cards */
```

## User Flows

### Assessment Flow
1. **Landing**: Click "Mulai Penilaian Baru" button
2. **Select Standard**: Choose RSPO or ISPO tab
3. **Select Farmer**: Dropdown with searchable farmer list
4. **Question Loop**:
   - Display indicator code and principle
   - Show question in Indonesian
   - Radio options: Ya/Sebagian/Tidak
   - Progress bar at top
   - Previous/Next navigation
5. **Completion**: Calculate score, save responses, show summary

### Traceability Entry Flow
1. **Click**: "Catat Transaksi" button
2. **Auto-generate**: Batch ID
3. **Input**:
   - Select buyer type
   - Enter buyer ID
   - Input quantity (kg)
   - Input price per kg
   - Toggle certification checkbox
   - If certified, select standard
4. **Calculate**: Auto-calculate total price
5. **Submit**: Save transaction, show in list

## Accessibility

### WCAG 2.1 Level AA Compliance

#### Color Contrast
- Text: Minimum 4.5:1 contrast ratio
- UI Components: Minimum 3:1 contrast ratio
- Current implementation meets standards

#### Keyboard Navigation
- All interactive elements focusable via Tab
- Focus rings visible: `ring-2 ring-primary ring-offset-2`
- Skip links for main content
- Arrow keys for dropdown navigation

#### Screen Reader Support
```tsx
// Semantic HTML
<nav aria-label="Main navigation">
<button aria-label="Close dialog">
<img alt="Descriptive text">

// ARIA attributes
aria-current="page"        // Active nav item
aria-expanded="true"       // Accordion state
aria-hidden="true"         // Decorative icons
role="alert"               // Error messages
```

#### Form Accessibility
- Labels always visible (never just placeholders)
- Error messages associated with inputs
- Required fields indicated
- Helper text for complex inputs

#### Icon + Text Pattern
```tsx
<Button>
  <Icon className="w-4 h-4" aria-hidden="true" />
  <span>Button Label</span>
</Button>
```
- Icons are decorative (`aria-hidden`)
- Text always present for screen readers
